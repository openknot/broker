broker
======

OpenKnot Broker.

Installation
------------

Either pull the automatically updated `Docker`_ image::
    
    $ docker pull openknot/broker

Or install from the development repository::
    
    $ git clone https://github.com/openknot/broker.git
    $ cd broker
    $ pip install -r requirements.txt
